## Contributing
### Installation

```shell
npm i
```

### Running locally

```shell
npm run storybook
```

### Deploying

The repo has been setup to use `gh-pages` to easily deploy to Github.
For the deploy to work you need to configure the `homepage` field in `package.json` to something like `http://{username}.github.io/{repo}`.

```shell
npm run deploy
```
